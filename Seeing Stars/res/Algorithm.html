You might have noticed that there's a general pattern you can follow to construct a star. It goes like this:

  1. Pick a number of points. Let's call that number p.
  2. Pick a step size. Let's call that number s.
  3. Draw p evenly-spaced points around the perimeter of a circle.
  4. Choose a point on the circle as your starting point, then draw a line to the point s points down the circle, moving clockwise.
  5. Repeat the previous step until you end up back where you started.

The star that you get by drawing things this way is called the { s / p } star. For example, the two stars that you drew in the previous sections were the { 7 / 2 } star and the { 7 / 3 } star.
